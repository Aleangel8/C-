Comandos basicos
	dotnet new
	dotnet restore
	dotnet build
	dotnet run
Instalar
	dotnet install package 


Crear Solucion
	dotnet new sln -n "Nombre" -o .
	dotnet sln list //para ver la lista de sln


Añadir Consola, WebApp y Libreria
	dotnet new console -n "ConsoleName" -o .\ConsoleName
	dotnet new mvc -n "WebAppName" -o .\WebAppName
	dotnet new classlib -n "Name.Core" -o .\NameCore

Añadir a solucion
	dotnet sln add/remove  // añadir o remover la sln
